# Simple Treatment Effect Notebook: HbA1c and Anemia

This is a beginner-friendly healthcare data science project that simulates a treatment comparison study.

## Project goal

The notebook compares **Treatment A** and **Treatment B** using two clinical outcomes:

- **HbA1c change** for diabetes control
- **Hemoglobin change** for anemia-related improvement

## What the notebook teaches

- how to simulate a healthcare dataset
- how to inspect and summarize data
- how to visualize treatment group differences
- how to run **Welch's t-test**
- how to run **Mann–Whitney U test**
- how to calculate **95% confidence intervals**
- how to interpret whether a treatment effect is meaningful

## Notes

- This project uses **simulated data** for learning purposes.
- For **HbA1c**, more negative change is better.
- For **hemoglobin**, more positive change is better.

## Learning focus

This notebook is designed for beginners who want to understand:

- what the code does
- why the code is used
- how to think through treatment-effect analysis instead of memorizing code
